React API Key Manager Live Demo

Discord

This site shows a live demonstration of the @zuplo/react-api-key-manager React control, that can be used in a developer facing portal or console. It allows users to self-serve their own API keys. In this case, the API Keys are managed by zuplo. However, the control is provider-based and you can create your own provider to support any API Key management service.

You can create a key below, use the control to manage your keys (try rolling them). At the bottom, you can test one of your keys which will call a /who-am-i endpoint, which will echo back the metadata associated with the key.

To try it out, follow our detailed walkthrough tutorial.
